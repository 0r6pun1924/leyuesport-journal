# leyuesport-journal

## 项目简介

`leyuesport-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，主要目的是收集、整理并公开呈现一系列静态 HTML 文档。

## 目录说明

```
.
├── index.html          # 项目入口页面
├── pages/              # 存放归档 HTML 页面的目录
│   ├── example1.html   # 示例页面
│   └── example2.html   # 示例页面
├── assets/             # 资源文件（CSS、JavaScript、图片等）
├── README.md           # 本文件
└── LICENSE             # 许可证文件
```

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，每个页面可单独访问。页面内容可能涵盖多种类型的信息，如文档、笔记、静态展示等。所有页面均未经过动态后端处理，仅依赖浏览器端渲染。

### 如何添加新页面

1. 在 `pages/` 目录下创建新的 HTML 文件。
2. 确保页面符合基本的 HTML 规范，并引用必要的资源（如 CSS、JS）。
3. 如有需要，在 `index.html` 中添加指向新页面的链接。
